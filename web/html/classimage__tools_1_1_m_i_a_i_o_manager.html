Detailed Description

The manager for I/O operations in MIA. This includes initializing GLUI control elements related to image loading/saving, as well as handling the load/save operations themselves. The load operation includes validating the image type and filesystem permissions before the load will proceed. It also handles loading an image from a file to the stamp.
