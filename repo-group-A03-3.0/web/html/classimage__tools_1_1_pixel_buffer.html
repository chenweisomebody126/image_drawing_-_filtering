Detailed Description

Stores an array of ColorData, such as an image that can be drawn to the screen. In this class, (0,0) is the top left corner of the screen/image, which is the not the same as the coordinate system defined by GLUT, where (0,0) is the bottom left corner of the screen. Your implementation will need to account for this.
